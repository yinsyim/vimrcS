### Installation ###

Install manually or with your favorite plug-in manager and then load the theme.

With `vim-plug`:

```VimL
Plug 'KKPMW/sacredforest-vim'
colorscheme sacredforest
```

To make the colorscheme work as intended either set `termguicolors`:

```VimL
set termguicolors
```
